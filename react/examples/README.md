# ⚛️ React Examples

이 폴더는 **React 학습 과정에서 작성한 기능별 예제 코드**를 모아둔 공간입니다.  
각 예제는 독립적으로 실행 가능한 형태로 구성되며, 학습 노트(`notes/`)와 연계하여 참고할 수 있습니다.  
React 기본 구성부터 상태관리, 이벤트 처리, 컴포넌트 설계 패턴까지 다양한 주제를 다룹니다.
